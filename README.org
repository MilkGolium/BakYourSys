* BakYourSys

此文档是 org-mode 文件，编写的时候需要注意格式，不要使用 Markdown 的语法。 \\
例如 *粗体* ，只需要一个星号，而不是 Markdown 的双星号，使用 ~#+begin_src~ ，而不是 Markdown 的 ~```~ 。 \\
注意修饰语法不要和全角字符挨在一起，中间需要加空格，例如 ~*粗体*示例~ 是错误的，应该写成 ~*粗体* 示例~ 。 \\
上下两段文本中间间隔一个空行即可分出段落，如果需要不分段落的换行，类似 HTML 的 ~<br>~ 标签，可以使用 ~\\~ ，这同样需要和全角字符间隔一个空格，例如 ~行尾 \\~ 。 \\
对于文档中的表格， Agent 在编写的时候只需要紧凑编写即可，不需要尝试匹配空格数量（这个工作交给 Emacs 来完成），例如 ~| A | B | C |~ ，而不是 ~|     A     |     B     |     C     |~ 。

** 项目简介

此项目与 Github MilkGolium/ConfigureYourSys 项目配套使用，且与之解耦。通过相通的数据库标准来协助完成备份和还原工作。 \\
这是一个系统备份工具，负责把系统中的软件以及配置文件录入到数据库和文件库里面，数据库可以存储自定义部署脚本，在还原的时候直接执行。
